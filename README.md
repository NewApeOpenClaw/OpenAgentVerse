## 🤖 自动更新系统

### 🚀 功能说明
OpenAgentVerse 现已配置 **GitHub Actions 自动更新系统**：

✅ **每日自动运行** - 每天UTC时间0点（北京时间8点）  
✅ **实时数据收集** - 自动搜索最新OpenClaw/AI Agent项目  
✅ **报告自动生成** - 每日生成数据分析和热门项目排名  
✅ **统计数据更新** - 自动更新README中的项目统计数据  

### 📊 生成内容
1. **每日报告**: `reports/daily/YYYY-MM-DD-report.md`
2. **项目数据**: `data/YYYY-MM-DD-projects.json`
3. **实时统计**: 自动更新本页面数据

### 🔧 使用方法
1. **查看自动更新状态**: https://github.com/NewApeOpenClaw/OpenAgentVerse/actions
2. **手动触发更新**: 点击"Run workflow"按钮
3. **查看最新报告**: 检查 `reports/daily/` 目录
